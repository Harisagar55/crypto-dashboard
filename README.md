# CRYPTO DASHBOARD

A real-time cryptocurrency dashboard built using React, Tailwind CSS, and CoinGecko API.  
This project provides live market data with a clean and responsive UI.

---

## 🚀 Features

- 📊 Live cryptocurrency prices
- 🔍 Search and filter by name or symbol
- 🌙 Dark / Light mode (saved using localStorage)
- 💱 Multi-currency support (USD, INR, EUR)
- 📈 Price trend charts (7-day)
- 🥇 Top gainers & losers
- 🔄 Auto-refresh every 60 seconds
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS
- CoinGecko API
- Recharts (for graphs)
